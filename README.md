# MSOLM : My Simple OLM

My Simple OLM is a php classes generator for mapping ldap directoies branches to php objects.
OLM stands for Object LDAP Mapping.
The generated classes don't need anything else to work and can be used as is.

The documentation is available at :

- EN : https://lezab.github.io/myPhp/en/base/msolm/
- FR : https://lezab.github.io/myPhp/fr/base/msolm/
